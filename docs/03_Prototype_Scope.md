## 1. Prototype description

The prototype demonstrates lighting system for hotel managers to configure lighting profiles and synchronize them across multiple different rooms. It is designed for users who need a fast and reliable way to manage lighting in large hotel environments.

The prototype allows creating lighting profiles, grouping rooms and applying configurations instantly to selected room groups. It also simulates how different room types can have different lighting setups.

The goal of the prototype is to prove the promise of the product “Configure once, sync everywhere”. Also it shows how the system saves time compared to manual room by room configuration.

## 2. Demonstration

| REQ-ID | Requirement description                                                  |
| ------ | ------------------------------------------------------------------------ |
| 001    | System allows creation of lighting profiles (brightness, color, scenes)  |
| 002    | Lighting profiles can be applied to multiple rooms simultaneously        |
| 003    | System supports grouping rooms (e.g. standard rooms, suites, restaurant) |
| 005    | Changes to profiles are synced instantly across selected rooms           |
| 008    | System works even when a hotel has big number of rooms                   |

These requirements were selected because they answer the VP’s questions:

- Time saving -> 002 and 005
- Different room types -> 003
- Large hotel support -> 008
- Easy to use -> 001 and 005
